# نسخة احتياطية: Deraah 1.0.1

## 📅 تاريخ النسخة الاحتياطية
15 أكتوبر 2025

## ✨ المميزات المتوفرة في هذه النسخة

### 1. تقرير الحضور والانصراف (AttendanceDashboard)
- تقرير كامل بتصميم احترافي
- عرض جدولي للحضور والانصراف
- **متجاوب بالكامل مع الجوال:**
  - عرض جدول على الكمبيوتر
  - عرض بطاقات على الجوال
- إحصائيات شاملة
- بحث وفلترة متقدمة
- تصدير CSV وطباعة

### 2. صفحة اختيار نوع المعاملة (TransactionSelectionPage)
- 12 نوع معاملة مختلفة
- بحث وفلترة حسب القسم
- **تصميم متجاوب:** عرض بطاقات مع تأثيرات
- تدرجات ألوان لكل نوع معاملة

### 3. نموذج طلب إجازة مرضية (SickLeaveForm)
- **نموذج كامل ومتقدم:**
  - معلومات الموظف (مملوءة مسبقاً)
  - تفاصيل الإجازة مع حساب تلقائي للمدة
  - المعلومات الطبية (التشخيص، الطبيب، المستشفى)
  - جهة الاتصال الطارئة
  - نظام رفع المرفقات (PDF, JPG, PNG)
- **Validation متقدم:**
  - التحقق من التواريخ
  - التحقق من المرفقات
  - رسائل خطأ واضحة
- **متجاوب بالكامل** مع جميع أحجام الشاشات
- تأثيرات حركية مع Framer Motion

### 4. نموذج طلب إجازة سنوية (AnnualLeaveForm)
- نموذج متكامل لطلب الإجازات السنوية
- حساب أيام الإجازة
- نظام البدائل

### 5. نظام حاضر (HazerSystem)
- تسجيل الحضور والانصراف
- عرض السجلات

## 🎨 الاستايل المعتمد

جميع الصفحات تستخدم نفس الاستايل الموحد:

```css
- خلفية: radial-gradient من السماوي والوردي
- بطاقات: backdrop-blur مع شفافية
- حدود: rounded-2xl
- ألوان: تدرجات ناعمة
- تأثيرات: hover و transitions سلسة
- responsive: breakpoints sm, md, lg
```

## 📚 المكتبات المستخدمة

- **React 18.3.1**
- **TypeScript 5.5.3**
- **Vite 5.4.2**
- **Tailwind CSS 3.4.1**
- **Framer Motion 12.23.12**
- **Lucide React 0.344.0** (أيقونات)
- **Radix UI** (مكونات UI)

### User Experience

- Loading states
- Error handling
- Form validation
- Success messages
