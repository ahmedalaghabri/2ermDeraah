# البوابة الإلكترونية - تصدير HTML + CSS + JavaScript

## 📁 ملفات المشروع

- `index.html` - الملف الرئيسي للصفحة
- `styles.css` - ملف التنسيقات والتصميم
- `script.js` - ملف الجافاسكريبت والتفاعل
- `README.md` - هذا الملف (دليل الاستخدام)

## 🚀 كيفية الاستخدام

1. **تحميل الملفات:**
   - قم بتحميل جميع الملفات في مجلد واحد
   - تأكد من وجود ملف الشعار `logonew.svg` في نفس المجلد

2. **فتح المشروع:**
   - افتح ملف `index.html` في أي متصفح ويب
   - أو استخدم خادم محلي مثل Live Server

3. **المتطلبات:**
   - متصفح ويب حديث (Chrome, Firefox, Safari, Edge)
   - لا يتطلب أي مكتبات خارجية

## 🎯 الصفحات المتاحة

1. **الرئيسية** - عرض جميع الخدمات والبطاقات
2. **إضافة معاملة** - نموذج شامل لإضافة معاملة جديدة
3. **تقرير الحضور** - إحصائيات وبيانات الحضور

## ⌨️ اختصارات الكيبورد

- `ESC` - العودة للصفحة الرئيسية
- `Ctrl + K` - التركيز على مربع البحث

## 🎨 التخصيص

### تغيير الألوان
يمكنك تعديل الألوان في ملف `styles.css` من خلال متغيرات CSS:

```css
:root {
    --accent-primary: #3b82f6;    /* اللون الأساسي */
    --accent-pink: #B21063;       /* لون الشارات */
    --bg-primary: #FAFCFF;        /* خلفية الصفحة */
}
```

### إضافة خدمات جديدة
يمكنك إضافة بطاقات خدمات جديدة في ملف `index.html`:

```html
<div class="service-card" data-action="new-service">
    <div class="card-gradient from-blue"></div>
    <div class="card-content">
        <div class="card-icon">
            <!-- أيقونة SVG هنا -->
        </div>
        <div class="card-info">
            <h3>اسم الخدمة</h3>
            <p>وصف الخدمة</p>
        </div>
    </div>
    <div class="card-action">
        <span>النقر للوصول</span>
    </div>
</div>
```

## 🔧 التطوير المتقدم

### إضافة صفحات جديدة
1. أضف محتوى الصفحة في `index.html`
2. أضف التنسيقات في `styles.css`
3. أضف منطق التنقل في `script.js`

### ربط بـ API
يمكنك ربط النماذج والبيانات بـ APIs خارجية من خلال تعديل الدوال في `script.js`.
